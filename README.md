# ProductStore
Implementation of a product store web application using Spring Boot and React.

## Server Dependencies
* Java SDK 21
* Spring Boot 3.3.0
* Maven 3.9.7
* Postgres version >= 10

### Client Dependencies
* Node v16.20.2
* Yarn 1.22.22
* Npm 8.19.4
* Axios
* Antd

## Installation
### Running the server:
    cd product-store-server 
    mvn clean package
    mvn spring-boot:run
    
### Runinng the client for the first time:
    yarn install
    
### Running the client:
    cd product-storeclient
    yarn start
